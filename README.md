# Golf Club Tracker ⛳

A mobile-optimised single-page web app for tracking which clubs you hit on each hole during a round of golf.

## Features

- Log clubs hit per shot, per hole across all 18 holes
- Clubs: D, 5W, 5H, 5i–9i, PW, PA, AW, SW, Putter
- Putter logged as a count (± stepper) rather than individual taps
- Shot sequence visible per hole with ability to remove individual shots
- Round summary showing all 18 holes with total shots and most-used club
- Copy full round to clipboard

## Usage

Open `index.html` in a mobile browser. For the best experience on iPhone:

1. Open the hosted URL in Safari
2. Tap **Share → Add to Home Screen**
3. Opens as a standalone app with no browser chrome

Single self-contained HTML file. No dependencies, no build step, no framework.
